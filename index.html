sFreeDroid by grezua

scala opengl(lwjgl) freeDroid resources

My play with scala + opengl. I'm going to use some content from freedroid project (http://www.freedroid.org/), as well as algorithms from their c sources.

scala 2.9. lwjgl >= 2.5
